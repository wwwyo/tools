# 五滴（goteki）

画像を貼り付けると、使用色の面積比 top5 を五滴抽出するツール。抽出した色は背景に「白い紙に滲んだインク」として散る。

## 使い方

- 画像をドラッグ&ドロップ / クリックで選択 / Cmd+V で貼り付け
- 「サンプル画像で試す」でページ内生成のサンプルを流せる
- HEX / RGB / CMYK をトグルで切り替え。各色 copy と一括 copy あり
- 解析はすべてページ内 Canvas で完結し、画像は外部に送信しない

## 色抽出ロジック（colorQuantize.ts）

1. **縮小サンプリング**: 長辺 200px に縮小して `getImageData`（最大約4万ピクセル）
2. **バケット量子化**: RGB 各チャンネルを 24 刻みで丸めて頻度カウント。alpha < 128 は除外
3. **代表色**: バケット中心値ではなく、バケット内実ピクセルの RGB 平均
4. **近似色マージ → top5**: 頻度順に走査し、既存クラスタと RGB ユークリッド距離 < 32 なら面積を合算（代表色は多数派を維持）。量子化の仕切り境界で割れた色を拾い直す後処理。最後に面積順 top5、ratio = count / 全ピクセル数

距離は知覚的でない素朴なユークリッド。厳密にやるなら Lab + ΔE だが、この用途では過剰なので採用していない。

## インク背景（main.ts renderBlobs）

レイヤーは `インク(z-0) → 白60%ウォッシュ(z-10) → コンテンツ(z-20)` の3層。

- **大きさ**: 各滴の面積 ∝ ratio（直径 ∝ √ratio）。top5 合計でビューポート面積の 25〜35% になるよう scale を逆算
- **位置**: 層化ランダム。ビューポートを色数分の縦の列に分割し、シャッフルした列に1色ずつ割り当てて横方向の偏りを防ぐ
- **形と質感**: 8値ランダムの border-radius で有機的な形にし、SVG フィルタ（feTurbulence + feDisplacementMap scale=120）で縁を滲ませる。色は中心 60% までベタで急に抜ける radial-gradient
- **重なり**: mix-blend-multiply でインクの重ね塗りのように沈む。opacity は面積比に応じて 0.5〜0.8
- **出現**: opacity 0 + scale(0.6) から 0.7s で遷移し、滴が落ちて広がる動き

## ファイル構成

```
goteki/
├── index.html        エントリ（title / description は一覧自動生成に使われる）
├── main.ts           UI・インク描画・コピー・フォーマット切替
├── colorFormat.ts    HEX / RGB / CMYK 変換
└── colorQuantize.ts  縮小・量子化・マージ・top5 抽出
```
